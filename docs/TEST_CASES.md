# CausaLab V1.0 测试用例与验证记录

## 1. 测试环境

- 操作系统：Windows
- 后端：Python 虚拟环境 `.venv`
- 前端：Node.js + npm
- 浏览器：本地开发浏览器
- 文档验证：LibreOffice + Poppler/PDF 渲染
