# CausaLab V1.0 产品与软著范围说明

## 1. 软件名称

- 全称：CausaLab 智能实验设计与效应评估平台
- 简称：CausaLab
- 版本：V1.0

所有界面、源代码页眉、课程报告、用户手册和软著材料统一使用以上名称与版本。

## 2. 解决的问题

平台帮助用户回答三个连续问题：

1. 实验开始前需要多少样本，才能检测到有意义的变化？
2. 实验结束后观察到的差异，是实际效应还是随机波动？
3. 分析结论是否受异常值、分布偏态、样本失衡等因素影响？

## 3. 目标用户

- 需要完成统计实验的学生和教师；
- 开展产品、教学或营销实验的数据分析人员；
- 希望在少量代码操作下获得规范统计结论的非专业用户。

## 4. V1.0 核心流程

创建分析项目 -> 规划实验 -> 导入数据 -> 质量诊断 -> 选择指标 -> 计算效应 -> 稳健性验证 -> 生成解释与报告。

## 5. 自主功能边界

核心统计流程、参数校验、结果解释规则、质量评分和报告组织由本项目实现。第三方库仅承担基础数值运算、Web 框架和图表渲染，不复制其他项目的业务代码或界面结构。

## 6. 课程要求映射

| 课程要求 | CausaLab 对应功能 |
|---|---|
| 用户调整参数 | 样本量、显著性水平、功效、置信水平、迭代次数 |
| 动态统计图表 | 功效曲线、分布图、箱线图、置信区间、Bootstrap 分布 |
| 实时计算结果 | 均值、方差、统计量、p 值、效应量和置信区间 |
| 文字解释 | 方法选择原因、结论、假设条件和风险提示 |
| 容错处理 | 字段校验、缺失值、重复记录、异常值和参数范围检查 |

## 7. 暂不纳入 V1.0

倾向得分匹配、因果图自动识别、贝叶斯 A/B 测试、序贯检验和异质性处理效应将在后续版本扩展，避免首版范围失控。
