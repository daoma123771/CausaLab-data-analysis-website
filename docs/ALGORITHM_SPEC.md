# CausaLab V1.0 核心算法说明

## 1. 实验规划模块

### 1.1 比例指标样本量

适用于转化率、通过率、点击率等二分类指标。系统输入对照组基准比例、希望检测到的目标比例、显著性水平、统计功效和实验组/对照组分配比例。

系统先使用反正弦平方根变换计算两比例的 Cohen's h：

```text
h = 2arcsin(sqrt(p1)) - 2arcsin(sqrt(p0))
```

随后基于独立两样本正态近似求解达到目标功效所需的每组样本量。计算结果向上取整，避免实际功效低于设定值。

### 1.2 连续指标样本量

适用于成绩、收入、停留时长等连续指标。根据最小均值差与标准差计算标准化效应量：

```text
d = 最小均值差 / 标准差
```

系统使用独立两样本 t 检验功效模型求解样本量，并支持非 1:1 分组。

### 1.3 功效曲线

系统以推荐样本量的 20% 至 160% 生成多个候选样本点，逐点计算检验功效，从而展示样本规模增加时发现真实效应的概率变化。

### 1.4 参数约束与提示

- 显著性水平限定在 0 至 0.5 之间；
- 目标功效限定在 0.5 至 0.999 之间；
- 比例指标必须位于 0 与 1 之间且基准值不能等于目标值；
- 标准差和最小均值差必须为正数；
- 对稀有事件、极小/极大标准化效应及严重不均衡分组给出风险提示。

## 2. 效应评估模块

### 2.1 比例指标

系统根据两组成功事件数和总样本量计算转化率差，使用合并方差的两比例 Z 检验计算 p 值。支持双侧、实验组更高和实验组更低三种备择假设。

### 2.2 连续指标

系统接收两组原始观测值，采用不要求方差齐性的 Welch 独立样本 t 检验，更适合真实业务数据。

### 2.3 Bootstrap 稳健性验证

- 比例指标按照观测比例进行参数 Bootstrap；
- 连续指标对组内原始样本进行有放回的非参数重抽样；
- 默认重复 3,000 次，以百分位法输出 95% 效应差置信区间；
- 前端展示重抽样分布直方图，同时报告效应大小、p 值、区间和自然语言决策结论。

### 2.4 决策规则

系统将 p 值与显著性水平比较，并检查观测效应方向是否符合单侧备择假设。输出“拒绝原假设”“暂不拒绝原假设”或“不支持预设方向”，避免把方向相反的显著结果误判为实验成功。

## 3. 数据质量诊断模块

系统支持 CSV 与 XLSX 文件，自动生成字段画像并推断标识符、实验分组、数值指标、时间、分类和文本角色。质量检查包括：

- 缺失单元格数量与比例；
- 完全重复数据行；
- 基于 1.5 倍四分位距（IQR）规则的数值异常值；
- 二元实验分组识别、样本占比和明显分组失衡；
- 根据完整性、唯一性、异常值率和分组平衡生成 0–100 综合质量分。

诊断仅提供统计提示，不会自动删除或修改用户数据；异常值仍需结合业务边界进行复核。

## 4. 分析报告模块

系统接收结构化实验结果，使用固定版式生成 DOCX 报告。报告包含项目元数据、执行摘要、核心指标表、置信区间、统计方法、判断规则、数据质量分、结论与风险备注。文件名进行非法字符过滤，报告数据通过 Pydantic 约束验证，以保证导出文档可复现且格式稳定。
