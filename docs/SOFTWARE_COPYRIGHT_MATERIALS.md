## 1. 软件基本信息

- 软件全称：CausaLab 智能实验设计与效应评估平台
- 软件简称：CausaLab
- 版本号：V1.0
- 软件类型：数据分析/统计实验教学辅助软件
- 运行方式：本地 Web 应用
- 开发语言：Python、TypeScript、Vue
- 主要框架：FastAPI、Vue 3、Vite、ECharts

## 2. 软件功能边界

CausaLab V1.0 的原创功能重点不在单一统计公式，而在“实验设计与效应评估工作流”的组织：

1. 实验前规划：通过参数化输入生成建议样本量和功效曲线。
2. 数据质量诊断：对上传数据进行字段识别、缺失/重复/异常值检测和分组比例判断。
3. 实验后评估：支持比例指标和连续指标的效应估计、假设检验和置信区间输出。
4. 稳健性验证：使用 Bootstrap 重抽样展示效应分布。
5. 报告生成：将实验元数据、统计结果、判断规则和结论自动组织为 Word 报告。

## 5. 答辩/说明建议

讲解时建议突出三点：

1. 输入输出清晰：输入实验参数或数据，输出样本量、效应量、p 值、置信区间、诊断建议和报告。
2. 工作流完整：覆盖实验前、实验中、实验后和报告沉淀，而不是只做一个统计检验。
3. 可复现性强：有接口、测试、算法文档、用户手册、示例数据和示例报告。
