# CausaLab V1.0 演示讲解脚本

## 1. 开场说明

本项目叫 CausaLab 智能统计分析与样本量评估平台 V1.0。它解决的问题是：用户上传一份 CSV/Excel 表格后，如何快速理解数据结构、判断适合做什么统计分析、样本量是否大致足够，并把分析过程沉淀成可复核报告。

## 2. 演示路径

### 第一步：工作台

介绍系统由连续分析流程组成：数据导入、数据画像、方法推荐、样本量判断、统计检验/数学建模和报告生成。说明它是一套面向表格数据的分析向导。

### 第二步：分析向导

进入“分析向导”，上传 `examples/housing_sample.csv`。展示系统自动识别 120 行、7 列数据，给出字段画像、目标变量候选、分组变量候选、质量评分和统计方法推荐。

讲解重点：用户不需要一开始知道该选 t 检验、相关分析还是回归，系统会先根据数据结构推荐可开展的分析路线。

### 第三步：样本量与方法推荐

查看“推荐统计分析路线”，说明系统会给出描述统计、相关分析、多组均值比较、线性回归等建议，并对每项分析给出样本量判断和适用条件。选择“建立连续目标变量的解释模型”，确认目标变量和解释变量后，点击“运行统计分析 / 建模”。

讲解重点：系统不仅推荐方法，还会直接执行建模，输出 R²、调整 R²、整体 p 值、回归系数、变量方向和解释文字。

### 第四步：专项统计工具

如果需要演示 A/B 实验，可以进入“效应评估”页面，输入对照组和实验组数据，展示 p 值、效应量、置信区间和 Bootstrap 分布。

讲解重点：新版主流程面向通用表格数据，旧版 A/B 检验能力作为专项分析工具保留。

### 第五步：报告导出

进入分析报告页面，填写项目名称、核心指标、统计结果和结论，导出 Word 报告。

讲解重点：报告中保留了参数、方法、结果和判断规则，方便课程提交和后续复查。

## 3. 结尾总结

CausaLab 的价值在于把统计学知识做成一个完整的软件流程。它不是简单调用某个统计库，而是围绕实验决策设计了输入校验、数据诊断、效应评估、稳健性验证和报告输出，因此更适合作为课程项目和软著雏形。
