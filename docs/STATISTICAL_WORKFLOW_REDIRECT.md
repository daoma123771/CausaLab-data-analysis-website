# CausaLab V1.0 统计分析向导改造方案

## 1. 改造目标

现有版本已经具备实验规划、数据诊断、效应评估和报告导出能力，但用户流程偏模块化。新版改造目标是将系统升级为连续的数据分析向导：

```text
上传表格数据 -> 数据画像 -> 选择分析目标 -> 推荐统计方法 -> 样本量与条件评估 -> 统计分析/数学建模 -> 结果解释 -> 报告归档
```

该方向保留现有统计计算能力，同时让软件更贴近真实用户的分析路径：用户通常先拥有一份 CSV/Excel 数据，然后希望知道这份数据能回答什么问题、适合使用什么方法、当前样本是否足够、分析结论是否可靠。

## 2. 软件定位

建议软件全称调整为：

```text
CausaLab 智能统计分析与样本量评估平台 V1.0
```

软件定位：

> 面向表格数据的统计分析向导工具。用户上传 CSV/Excel 数据后，系统自动识别字段类型与数据质量，结合用户选择的分析目标推荐统计方法，评估样本量是否满足目标分析要求，并输出统计检验、建模结果、解释文本和可复核报告。

## 3. 用户输入

第一阶段重点支持常见二维表格数据：

- CSV 文件；
- Excel `.xlsx` 文件；
- 一行代表一个样本或观测对象；
- 一列代表一个变量；
- 字段可以包含数值型、分类型、时间型、ID 型和文本型。

适合的演示数据包括：

- 房价数据：目标变量为房价，解释变量为面积、收入、房龄、位置等；
- 工资数据：目标变量为工资，解释变量为学历、行业、工龄、地区等；
- 汽车油耗数据：目标变量为油耗，解释变量为重量、马力、年份等；
- A/B 实验数据：分组变量和转化/指标变量。

## 4. 第一阶段支持的分析问题

| 分析问题 | 数据结构 | 推荐方法 |
|---|---|---|
| 一个数值变量的分布情况如何 | 单个数值变量 | 描述统计、分布诊断、异常值检测 |
| 两个数值变量是否相关 | 两个数值变量 | Pearson/Spearman 相关分析 |
| 两组均值是否不同 | 一个二分类分组变量 + 一个数值变量 | Welch t 检验、效应量、置信区间 |
| 多组均值是否不同 | 一个多分类分组变量 + 一个数值变量 | 单因素 ANOVA |
| 一个连续目标受哪些因素影响 | 连续目标变量 + 多个解释变量 | 多元线性回归 |
| 转化率或比例是否存在组间差异 | 二分类分组变量 + 二元结果变量 | 两比例 z 检验 |

## 5. 样本量与条件评估逻辑

样本量是否“足够”不能脱离分析目标，需要结合：

- 显著性水平 `alpha`，默认 0.05；
- 目标统计功效 `power`，默认 0.80；
- 最小可检测效应 `MDE` 或标准化效应量；
- 当前可用样本量；
- 分组数量和每组样本量；
- 缺失值、异常值和分组失衡情况。

系统第一阶段输出三类结论：

1. 样本量充足：当前数据满足默认检验要求；
2. 样本量偏少：提示建议最低样本量与缺口；
3. 条件不适合：例如缺少目标变量、分组变量类别过多、有效样本过少或缺失严重。

## 6. 方法推荐规则

系统不要求用户一开始知道统计方法，而是先选择分析目标。推荐规则示例：

```text
如果用户选择“分析数值目标受哪些因素影响”
且目标字段是数值变量
且至少存在 1 个可用解释变量
则推荐“多元线性回归”

如果用户选择“比较两组是否不同”
且分组字段有 2 个取值
且指标字段为数值变量
则推荐“Welch t 检验”

如果用户选择“查看两个变量是否相关”
且两个字段均为数值变量
则推荐“相关分析”
```

## 7. 软著友好创新点

软件著作权材料中不应强调“发明统计公式”，而应强调软件实现和流程组织：

1. 表格数据结构自动画像；
2. 面向用户问题的分析目标向导；
3. 基于字段类型和数据结构的统计方法推荐；
4. 样本量适配与缺口提示；
5. 统计检验与数学建模结果解释；
6. 可复核 Word 报告自动生成；
7. 本地化、轻量化、低门槛的分析流程。

## 8. 当前实现边界

当前已完成：

- 新增“分析向导”页面；
- 上传 CSV/Excel 后自动生成数据画像；
- 推荐可用分析任务；
- 输出候选目标变量、分组变量、数值变量；
- 输出样本量与数据条件初步判断；
- 给出可解释的方法推荐列表；
- 支持在同一页面直接运行描述统计、相关分析、Welch t 检验、ANOVA、两比例 z 检验和多元线性回归；
- 输出指标卡、统计结果表、解释文本和风险提示。

后续可继续增强：

- 通用统计分析报告导出。
- 交互式图表；
- 回归诊断图、残差分析和多重共线性检测；
- 更多数学建模方法，如 Logistic 回归、聚类分析和主成分分析。
